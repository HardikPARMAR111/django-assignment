Django Quiz Application
A quiz management system built with django allows users to take quizzes and administrators to create and manage quiz content.
Features

User Features

Browse Available Quizzes - View all published quizzes with descriptions
Take Quizzes - Answer multiple-choice questions with an intuitive interface
View Results - See detailed results with correct/incorrect answers highlighted
Upcoming Evnts - can see all the upcoming events

Admin Features

Create Quizzes - Add new quizzes with titles and descriptions
Add Questions - Create questions with multiple answer choices
Mark Correct Answers - Designate the correct answer for each question
Edit Quizzes - Update quiz information
Delete Quizzes - Remove quizzes and associated data
Submissions Dashboard - View all user submissions with statistics

Technology Stack

Backend: Django 5.1.2
Frontend: HTML5, Tailwind CSS
Database: SQLite
Python: 3.12.0
